POWERLIFTING PROGRESSION - website
==================================

To view:
  Just open index.html in any web browser (double-click it).
  It is a single self-contained file - all styles, scripts and fonts
  load inline or via the Google Fonts CDN, so no other files are needed
  (an internet connection is used only to fetch the fonts).

Optional - run a local server:
  Install Node.js, then in this folder run:  node serve.mjs
  Then open http://localhost:3020
